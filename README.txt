## 💬 Sobre o projeto

O FURIA Fan Chat é uma aplicação web feita em React que simula uma comunidade de fãs da equipe de CSGO FURIA. A ideia é criar um ambiente divertido e informativo, onde o usuário pode

- Conversar com um bot da comunidade;
- Ver os próximos jogos da FURIA;
- Acompanhar notícias;
- Conferir a line-up do time.

Tudo isso em uma interface leve, responsiva e com uma pegada torcedor raiz.  

---

## ✨ Funcionalidades
  
✔️ Informações sobre partidas (data, campeonato, horário e placar)  
✔️ Seção de notícias atualizadas da FURIA  
✔️ Visualização do elenco atual (nome, nick e função)  
✔️ Rolagem automática no chat  
✔️ Interface mobile-friendly  
✔️ Ícone de notificação animado  

---

## ⚙️ Tecnologias

- React.js
- Hooks (`useState`, `useEffect`, `useRef`)
